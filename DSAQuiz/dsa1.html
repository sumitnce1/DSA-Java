Linked List Quiz

1. What is a linked list?

a) A linear data structure
b) A non-linear data structure
c) A hierarchical data structure
d) A combination of data structures
Correct Answer: a

2. Which of the following is the advantage of a linked list over an array?

a) Constant-time access to elements
b) Memory efficiency
c) Fast search operations
d) Automatic resizing
Correct Answer: b

3. In a singly linked list, each node contains:

a) Only the data
b) Only the reference to the previous node
c) Only the reference to the next node
d) Both the data and the reference to the next node
Correct Answer: d

4. In a doubly linked list, each node contains:

a) Only the data
b) Only the reference to the previous node
c) Only the reference to the next node
d) Both the reference to the previous node and the next node
Correct Answer: d

5. Which operation is not possible in a singly linked list?

a) Traversing from the last node to the first node
b) Inserting a new node after a given node
c) Deleting a node without traversing the list
d) Finding the middle element of the list in one pass
Correct Answer: a

6. What is the time complexity to insert a node at the end of a singly linked list?

a) O(1)
b) O(log n)
c) O(n)
d) O(n log n)
Correct Answer: c

7. Which of the following operations requires O(n) time complexity in a doubly linked list?

a) Insertion at the beginning
b) Deletion from the end
c) Traversing from the first node to the last node
d) Searching for a specific element
Correct Answer: d

8. The process of combining two sorted linked lists into a single sorted list is called:

a) Merging
b) Splitting
c) Sorting
d) Joining
Correct Answer: a

9. Which of the following is a drawback of using linked lists?

a) High memory utilization
b) Limited operations available
c) Inefficient for searching elements
d) Difficulty in implementation
Correct Answer: c

10. Which type of linked list contains a loop or cycle in its structure?

a) Circular linked list
b) Doubly linked list
c) Singly linked list
d) Unidirectional linked list
Correct Answer: a
